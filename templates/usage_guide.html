Usage Guide


Nearest Bus Stop

Search for the nearest bus stop to a point on the map

Steps
1 Click on a point on the map to select that point. A popup with the Latitude and Longitude of the point will appear
2 Click on the submit button
3 The details of the closest bus stop to the point is displayed, with the buses that pass through it
4 The map image shows the straight line path of the selected point (A) and the nearest bus stop (B)

Pathfinder

Search for the optimal route between two bus stops

Steps
1 Select the source and target bus stops using drop down inputs. Type into the input to search for a bus stop by its description or code
2 Check the checkbox to allow walking (up to 300m) in the route found.
3 Select the criteria to find a route, either by "Best Route", where transfers are discouraged but used if it significantly reduces the distance travelled or "Minimize Transfers" where transfers are heavily discouraged and used only if necessary
4 The algorithm takes some time to calculate the routes. The time taken increases depending on the number of bus services originating from the source bus stop and terminating at the target bus stop
5 All routes with the shortest distance are shown. Walking distances are added to the total distance, but are twice that of the actual walking distance.

 
 

Bus Stop Lookup

View details of a bus stop and all bus services passing through it

Steps
1 Select the bus stop using drop down input. Type into the input to search for a bus stop by its description or code
2 Click on the submit button
3 The details of the bus stop is displayed, with the buses that pass through it. A map is also displayed, showing its surroundings

Bus Service Lookup

View details of a bus service and the route it serves

Steps
1 Select the bus service using drop down input. Type into the input to search for a bus service by its number
2 Click on the submit button
3 The details of the bus service is displayed, with the routes that the bus operates

Additional Info

1 Bus Stop Codes displayed in pathfinder and bus service lookup are clickable to view details of the bus stop
2 Bus Services displayed in bus stop details and pathfinder are clickable to view details of the bus service
